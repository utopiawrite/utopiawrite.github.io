# LEAP Call 出場決策工具

這是一個純瀏覽器工具，幫助記錄和分析 LEAP Call 倉位的出場時機。

👉 [開啟工具](https://utopiawrite.github.io/leap.html)

## 使用說明

- 在此輸入的資料只存在你自己的瀏覽器，其他人看不到，也不會互相影響
- 資料不會跨裝置或跨瀏覽器同步，建議固定使用同一台裝置、同一個瀏覽器來記錄
- 如需換裝置，請先使用工具內的「備份」功能下載 JSON 檔，再到新裝置「匯入備份」
